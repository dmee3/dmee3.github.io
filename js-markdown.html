Javascript Markdown Engine

Syntax

  • Inline Text transformations
  • These tags can be placed around text to apply their rule to all content inside the tag.
  • Placing just a single tag will apply the rule up to the end of the line.
    • Bold: *
    • Italic: ^
    • Underline: _

  • Inline Elements
  • These elements can be mixed with plain text on a single line
    • Hyperlink: [Link text](http://www.website.com)
  • Line-Level Elements
  • These elements must be located on a line with no other text.
    • Horizontal Rule: ==
    • Headers: All text on the line will become a header
      • # H1
      • ## H2
      • ### H3
      • #### H4
      • ##### H5
      • ###### H6

Escape symbols by preceding with \

For example: *This* is bold but \*this\* is not = This is bold but *this* is not
