Javascript Markdown Engine

Syntax

To bold, underline, or italicize text, surround it by the appropriate characters. Any unmatched text-modification character will apply its rule up to the end of the line (for example, *abc = abc)

  • Bold: *
  • Italic: ^
  • Underline: _
  • Hyperlink: [Link text](http://www.website.com)

Escape symbols by preceding with \

For example: *This* is bold but \*this\* is not = This is bold but *this* is not
