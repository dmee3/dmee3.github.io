Cap City Website

September 2016

The Cap City website serves as both a public-facing informational website serving static content, and a web app to help run Cap City Percussion. Members can log in to see season schedules, add conflicts with upcoming rehearsal dates, and pay membership fees.

The website is hosted on a Digital Ocean droplet set up as a LAMP stack. Laravel is used as a back-end framework for all server-side code. Google's Material Design specification served as a guide for the UI, and MaterializeCSS was used as the foundation for the site's style. Additionally, VueJS was employed on certain pages to serve and format large amounts of data with minimal code repetition.

iMusic

November 2014

iMusic is an iOS music player with added Facebook and YouTube integration. Aside from playing music stored on the device, users can view a list of Facebook friends who use the app, see what their friends have been listening to, and browse/search YouTube to find music videos.

The application was developed for iOS by a 3-man team for Ohio State's Mobile Application Development course. I and my teammates distributed the workload at regular intervals based on an agile development plan. My most important contribution to the project was the implementation of a custom class to keep track of audio playback, song queues, and other information related to the music player itself.
