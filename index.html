Stacker


Completed December 2014


Stacker is a cross-platform mobile application for scheduling group meetings. It was developed using Adobe PhoneGap to provide support for multiple mobile devices and jQuery Mobile to facilitate a fast and responsive mobile design. The backend was built on a traditional LAMP stack with server space provided by the Ohio State Computer Science Engineering department during development.


Though my team has finished our contribution, the application is still in development. Further details will follow once Stacker has been published.


Stacker was created by a four-person team as part of a Capstone course in Software Applications. My primary contribution to the development process was writing code to communicate with 3rd party calendar services such as Google Calendar. As the project drew to a close, I branched out to work on interfacing between mobile devices running our software and the central Stacker server, as well as improving the UI for a more fluid experience.

iMusic


Completed November 2014


iMusic is an iOS music player with added Facebook and YouTube integration. Aside from playing music stored on the device, users can view a list of Facebook friends who use the app, see what their friends have been listening to, and browse/search YouTube to find music videos.


The iMusic application was developed for iOS by a 3-man team for Ohio State's Mobile Application Development course. I and my teammates distributed the workload at regular intervals based on approximately bi-weekly project status checks throughout the semester. My most important contribution to the project was the implementation of a custom music player class which kept track of song information, as well as ordered and randomized queues of songs based on user input. I also designed logos for the application, created the player interface view, and worked on standardizing the UI for easy scalability as more features were added to the app.
